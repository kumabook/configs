# My config files

Currently, there are following config files.

* init.el (for emacs)
* .screenrc (for gnu screen)
* .zshrc (for zsh)
* .pentadactylrc (for pentadactyl)
* .pentadactyl/colors/
* .pentadactyl/plugins/
